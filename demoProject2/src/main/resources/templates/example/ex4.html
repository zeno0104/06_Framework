삼항 연산자

      - 타임리프 속성 (th:)에
      삼항 연산자(조건식 ? 참 : 거짓) 작성 가능

삼항연산자

Elvis 연산자 ( ?: )

      [작성법]
      값 ?: 값이 없을 때

      - 삼항 연산자에서 조건식 자리에 값(변수)만 작성
      ( == , != 등의 연산자 사용 X)

      - 왼쪽 값이 존재하면 해당 값을 출력, 없으면 오른쪽 값이 출력

No-Operation 연산자 (값 ?: _ )

      - 왼쪽 값이 없을 경우 타임리프 코드를 해석하지 않는 연산자에서
      - 타임리프 코드 해석 X
      -> 일반 HTML 태그로 동작
      -> HTML 태그 사이 내용이 화면에 출력
